DFS Pro Engine

A Streamlit web app for generating optimized Daily Fantasy Sports (DFS) lineups from a CSV upload.

Features

Upload player CSV files

Normalizes different CSV column formats automatically

Generates DFS lineups based on salary cap and projections

Supports multiple lineup generation
# DFS Pro Engine
